This package can handle the user registration from different sources (Facebook, Google, Github, etc.).
To extend a user attributes the package create a profile table for the extra information.

## Install

Via Composer

``` bash
$ composer require laravel-issue-tracker/user
```

## Usage

``` php
$request = Request::create('/api/v1/users', 'GET', [api_token' => 'Your personal API token']);
\Request::replace($request->input());
$response = \Route::dispatch($request);

echo $response->name;
```
